# Kiểm tra trạng thái hệ thống (System Status Check)

Kiểm tra sức khỏe hệ thống trên tất cả các dịch vụ:

1. Truy vấn trạng thái pod Kubernetes
2. Kiểm tra các kết nối cơ sở dữ liệu
3. Giám sát thời gian phản hồi của API
4. Xem xét tỷ lệ lỗi
5. Kiểm tra mức độ sử dụng tài nguyên
6. Báo cáo sức khỏe tổng thể
