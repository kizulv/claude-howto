# Alert Analyzer

Phân tích sức khỏe hệ thống và các cảnh báo:
- Tương quan các cảnh báo (Alert correlation)
- Phân tích xu hướng
- Xác định nguyên nhân gốc rễ (Root cause)
- Trực quan hóa các chỉ số
- Phát hiện vấn đề một cách chủ động
